# HTML / CSS TIKTOK REEL
This project is the fourth exercise in the context of a Web Development Master Course. It is focused on HTML and CSS, including CSS Flexbox, and a more advanced focused on object positioning.

## TASK
The task is re-creating a web-page, provided by the teacher, by using HTML and CSS. This webpage is a Tik Tok Reel page. The objective is to obtain a finalized page with responsive layout focused on structural CSS and clean HTML architecture:
- Focus: Flexbox/Grid and semantic HTML. The approach used is Mobile-First.
- Bonus tasks:
1. add underline effect under "Per te", exactly as shown in the reference screenshot.
2. add red dot on the top left of "Seguiti", in the correct position.
3. add "plus / add" simbol below user-picture on the sidebar, in the correct position.
4. add number of messages to read on the top left of the message icon in the footer, in the correct position.
5. obtain the same effect on the border of the "plus / add" button in the footer, exactly as shown in the reference screenshot.
